# Release Checklist

Before marking a Codex task complete:

- [ ] App builds
- [ ] No TypeScript errors
- [ ] No broken imports
- [ ] No duplicate routes
- [ ] No exposed API keys
- [ ] API routes return JSON
- [ ] Supabase access is user-scoped
- [ ] UI works on mobile
- [ ] Changed files are listed
- [ ] Manual test steps are documented
- [ ] Any risks are documented
- [ ] Any follow-up tasks are documented
