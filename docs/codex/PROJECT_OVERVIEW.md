# Personara.ai Project Overview

## Platform Positioning
Personara.ai is a strengths-based intelligence platform that builds AI around who people naturally are, not just what they type.

Most AI tools are fast but generic. Personara.ai is personally fitted, context-aware, and designed to turn a person's natural strengths into practical action across work, career, family, teams, and leadership.

## Core Product Modules

### Persona Foundry
Creates a GPT-style personality tailored to the user's Gallup strengths and working style.

Primary outcomes:
- More useful personal AI behaviour
- Better communication fit
- Lower prompting friction
- Higher trust and repeat use

### Career Intelligence
Turns a user's strengths, CV, LinkedIn profile, work history, and career goals into practical career assets.

Primary outcomes:
- Executive positioning
- CV improvement
- Cover letters
- Interview guidance
- Job-search strategy
- Role-fit analysis

### TeamSync
Maps the interacting strengths of groups.

Primary audiences:
- Executive teams
- Boards
- Families
- Couples
- Project teams
- Gallup practitioners

Primary outcomes:
- Better collaboration
- Reduced friction
- Scenario testing
- Group prompts
- Team language
- Practitioner-supported recurring revenue models

## Strategic Thesis
Identity becomes infrastructure for growth.

Personara.ai uses a shared Gallup-based core engine across the platform so each module becomes stronger as the platform understands the user, team, or organisation better.

## Technical Stack
- Next.js App Router
- TypeScript
- Supabase Auth
- Supabase Database
- Tailwind CSS
- OpenAI API
- Server-side API routes under `/app/api`

## Build Principles
- Keep the platform modular.
- Avoid duplicate implementations.
- Keep AI logic server-side.
- Keep user data protected.
- Avoid large rewrites.
- Prefer reusable services and typed helpers.
- Build for future integration with practitioners, recruiters, job boards, LinkedIn-style platforms, and enterprise teams.
